#recipe-sharing-challenge-java

You can run test and application from root recipe-sharing using:
```
./mvnw test
```
For integration test you need switch on docker for postgres testcontainer.
You can turn off such testes by "testcontainers.enabled: false"
```
./mvnw clean spring-boot:run
```

Documentation is generated by Swagger 3.0. You can find descriptions dto and controller endpoints here:
```
http://localhost:8080/swagger-ui/index.html#/
```
